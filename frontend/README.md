# CRI Selvazzano Dentro - Frontend

Sito web moderno per il Comitato di Selvazzano Dentro della Croce Rossa Italiana.

## Installazione

```bash
cd frontend
npm install
```

## Sviluppo

```bash
npm run dev
```

Il sito sarà disponibile su http://localhost:3000

## Build

```bash
npm run build
```

## Tecnologie

- Vue 3
- Vue Router
- Vite
- CSS moderno con variabili

## Struttura

- `/src/views` - Pagine principali
- `/src/components` - Componenti riutilizzabili
- `/src/data` - Dati JSON importati da WordPress
- `/src/assets` - Immagini e CSS
